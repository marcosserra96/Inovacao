# App — Dinâmica de Inovação

Frontend React + TypeScript + Vite da plataforma. Instalação, configuração do Supabase e roteiro de teste ponta a ponta estão no [README da raiz do projeto](../README.md).

```bash
npm install
cp .env.example .env   # preencha com os dados do seu projeto Supabase
npm run dev
```
